# Caso_arboles

Este código implementa un formulario de Windows Forms que permite trabajar con un árbol visual usando un `TreeView`. El usuario puede agregar nodos raíz o hijos, buscar un nodo por su texto, recorrer el árbol en preorden, inorden o postorden, contar el total de nodos y calcular la profundidad máxima del árbol. Todas estas operaciones se realizan mediante botones que ejecutan métodos recursivos para insertar, buscar, recorrer y calcular datos del árbol. Además, el programa valida la entrada del usuario y muestra los resultados o errores mediante `MessageBox`.

#Grafo

Este código implementa un formulario que permite crear y administrar un grafo no dirigido donde los edificios son nodos y las conexiones entre ellos representan aristas. Utiliza un diccionario con listas de adyacencia para almacenar la estructura del grafo, permite añadir edificios y conexiones mediante entradas de texto, muestra todas las aristas en un DataGridView, comprueba si el grafo está completamente conectado mediante un recorrido y también calcula la ruta más corta entre dos edificios.
